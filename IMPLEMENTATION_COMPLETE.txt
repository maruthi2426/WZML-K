HOW IT WORKS NOW
================================================================================

COMMAND FLOW:

User Input:
  /leech https://vegamovies.nf/kalinga-2024 -q 1080p
       ↓
mirror_leech.py:
  1. Parses arguments
  2. Extracts quality: "-q" → "1080p"
  3. Stores self.quality = "1080p"
  4. Calls direct_link_generator(link, quality)
       ↓
direct_link_generator():
  1. Detects "vegamovies" in URL domain
  2. Routes to webscrapper_handler(link, quality)
       ↓
webscrapper_handler():
  1. Loads VegamoviesScraper from bot/scraper/vg.py
  2. Creates scraper instance
  3. Calls scraper.scrape(link, "1080p")
       ↓
VegamoviesScraper.scrape():
  1. Fetches page with requests.get(url)
  2. Parses HTML with BeautifulSoup
  3. Finds all shortener links
  4. Normalizes quality parameter
  5. Filters links by quality (1080p)
  6. Resolves shorteners to direct URLs
  7. Returns results with metadata
       ↓
Direct Link Found:
  [{"url": "https://direct-link...", "quality": "1080P", "size": "1.2GB"}]
       ↓
Bot Output:
  [LOG] Link 1: Kalinga - 1080P - 1.2 GB
  [LOG] URL: https://video-downloads.googleusercontent.com/...
       ↓
Bot Action:
  Starts download immediately

TOTAL TIME: 5-10 seconds

QUALITY FILTER IMPLEMENTATION
================================================================================

PARSING:
  Command: /leech url -q 1080p
  Extract: "-q" flag → value "1080p"
  Store: self.quality = "1080p"

NORMALIZATION:
  Input: "1080p", "1080P", "1080 p", "HD 1080p"
  Output: All normalized to "1080P"
  Logic: Uppercase + remove spaces

FILTERING:
  Step 1: Find all shortener links on page
  Step 2: For each link, extract quality info
  Step 3: Match against normalized filter
  Step 4: Keep only matching quality links
  Step 5: Resolve first matching link

SUPPORTED QUALITIES:
  ✅ 480p - Standard Definition
  ✅ 720p - HD Quality
  ✅ 1080p - Full HD (Most common)
  ✅ 2160p / 4k - 4K Quality
  ✅ Omit -q for all available
