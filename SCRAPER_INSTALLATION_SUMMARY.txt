HOW TO USE
================================================================================

Basic Syntax:
   /leech <vegamovies_url> -q <quality>

Examples:
   /leech https://vegamovies.nf/37740-kalinga-2024-hindi-dual-audio-web-dl-1080p-720p-480p.html -q 1080p
   /leech https://vegamovies.nf/movie-link -q 720p
   /leech https://vegamovies.nf/series-link (no quality filter, gets all)

Quality Options:
   - 480p (SD Quality)
   - 720p (HD Quality)
   - 1080p (Full HD Quality)

================================================================================
TROUBLESHOOTING QUICK REFERENCE
================================================================================

Issue: "ERROR: WebScrapper module not loaded"
→ Check bot/scraper/vg.py exists
→ Verify direct_link_generator.py is updated
→ Restart the bot

Issue: Chrome WebDriver errors
→ Run: pip install --upgrade webdriver-manager
→ Or: apt-get install chromium-browser chromium-chromedriver

Issue: No links found
→ Verify URL is accessible
→ Check quality spelling (1080p vs 1080P)
→ Wait a few seconds (page rendering takes time)

Issue: Slow scraping
→ This is normal - JavaScript rendering takes 5-10 seconds per URL
→ Multiple quality options = longer wait times

For more help, see: bot/scraper/README.md
